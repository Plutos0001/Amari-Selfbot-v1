# Amari-Selfbot
If you need to promote your server/socials, I recommend using amari. It has a built-in Mass DM

# Noteable Features
- auto-bump (helps u promote ur server ez)
- snipe
- editsnipe
- antinuke
- musicplayer
- message logger
- mee6 auto level up
- nitro sniper
- giveaway sniper
- slotbot sniper
- image search
- image manipulation
- massreact
- copycat
- 24/7 yui kiss/hug
- many more

# Installation

Go to config.txt and paste your token, prefix, password and if you want nitro sniper or not to be on!

install requirements.txt

pip install -r requirements.txt

Open amari.py and enjoy!

(if it asks to turn off windows defender, do it so that the selfbot can run)
